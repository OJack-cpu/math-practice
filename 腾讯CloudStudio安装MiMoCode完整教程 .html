第一章：基础知识介绍

1.1 Node.js 是什么？

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时环境，让 JavaScript 可以在服务器端运行。它使得开发者可以用 JavaScript 编写后端服务、命令行工具等。

主要特点：

  • 基于事件驱动、非阻塞 I/O 模型
  • 自带 npm 包管理器
  • 跨平台支持（Windows、macOS、Linux）

验证安装：

node -v # 查看 Node.js 版本 npm -v # 查看 npm 版本

1.2 npm 是什么？

npm（Node Package Manager）是 Node.js 的官方包管理器，用于安装、管理和分享 JavaScript 包（也称为模块）。

常用命令：

npm install # 安装包 npm install -g # 全局安装包 npm list -g # 查看全局安装的包 npm update # 更新包

全局安装 vs 本地安装：

  • 全局安装（-g）：包安装在系统目录，所有项目都能使用
  • 本地安装：包安装在当前项目的 node_modules 目录，仅当前项目可用

1.3 MiMo Code 是什么？

MiMo Code 是小米推出的终端原生 AI 编程助手，可以在终端中读写代码、运行命令、管理 Git，并通过持久化记忆系统跨会话理解你的项目。

核心功能：

多 Agent 模式

build（开发）、plan（分析）、compose（编排）

持久化记忆

跨会话记忆项目上下文

智能上下文管理

自动保存和恢复会话状态

任务跟踪

树形任务系统

语音输入

支持实时语音转文字

第二章：CloudStudio 环境准备

步骤 1：登录 CloudStudio

  1. 访问 https://cloudstudio.net
  2. 使用腾讯云账号登录（支持微信扫码、QQ、账号密码等方式）
  3. 登录后进入 CloudStudio 控制台

步骤 2：创建工作空间

  1. 点击「新建工作空间」或「创建应用」
  2. 选择 「All in One」 模板（或其他包含 Node.js 的模板）
  3. 配置工作空间：
    • 名称：自定义一个名称，如 mimo-code-workspace
    • 地域：选择离你最近的区域
    • 规格：根据需求选择 CPU 和内存配置
  4. 确认创建，等待环境初始化完成
注意：All in One 模板已内置 Node.js 环境，无需额外安装。

步骤 3：验证 Node.js 环境

在终端中运行以下命令，确认 Node.js 已安装：

node -v # 例如输出：v18.x.x npm -v # 例如输出：9.x.x

如果返回版本号，说明 Node.js 环境已就绪。

如果没有安装 Node.js，可以使用 nvm 手动安装：
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash source ~/.bashrc nvm install --lts

第三章：安装 MiMo Code

方式一：使用一键安装脚本（推荐）

在 CloudStudio 终端中运行：

curl -fsSL https://mimo.xiaomi.com/install | bash

方式二：使用 npm 安装

npm install -g @mimo-ai/cli

验证安装

mimo --version

启动 MiMo Code

mimo
首次启动时，MiMo Code 会自动引导你完成配置。

第六章：常见问题

Q1：CloudStudio 的积时用完怎么办？

CloudStudio 提供新用户 50 积时免费额度，用完后可考虑：

  • 等待下次登录刷新
  • 查看是否有其他免费活动

Q2：npm 安装权限报错（EACCES）？

# 方案一：使用 sudo sudo npm install -g @mimo-ai/cli # 方案二：使用 unsafe-perm npm install -g @mimo-ai/cli --unsafe-perm # 方案三：修改 npm 全局目录（推荐） mkdir ~/.npm-global npm config set prefix '~/.npm-global' export PATH=~/.npm-global/bin:$PATH

Q3：mimo 命令找不到？

确保 npm 全局安装目录在 PATH 中：

npm config get prefix # 查看全局安装路径 # 将输出路径下的 bin 目录添加到 PATH

Q4：网络问题导致安装失败？

设置 npm 镜像源：

npm config set registry https://registry.npmmirror.com npm install -g @mimo-ai/cli

Q5：WSL 下剪贴板乱码？

sudo apt install xsel

Q6：CloudStudio 终端没有 Node.js？

使用 nvm 手动安装：

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash source ~/.bashrc nvm install --lts

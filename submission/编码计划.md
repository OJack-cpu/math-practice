### 阶段二：SM-2 算法核心实现
**目标**：实现间隔重复调度算法

- [x] `calculate_quality()` — 根据正确性和答题时间计算质量评分（0~5）
- [x] `sm2_update()` — 核心算法：更新 EF、interval、repetitions
- [x] 算法演示脚本（模拟多次复习场景）
- [x] 6 种质量评分边界测试
- [x] EF 下限保护（≥1.3）

**SM-2 算法公式**：
```
quality < 3:     repetitions = 0, interval = 1
quality >= 3:
    rep=0 → interval=1
    rep=1 → interval=6
    rep>1 → interval = round(interval * EF)
EF' = EF + (0.1 - (5-q) * (0.08 + (5-q) * 0.02))
EF' = max(EF', 1.3)
```

## 三、技术难点与解决方案

| 难点 | 解决方案 |
|------|---------|
| LaTeX 公式渲染 | 集成 KaTeX 库，创建 MathText 组件自动识别 $...$ 和 $$...$$ |
| SM-2 算法实现 | 独立模块化设计，20个单元测试覆盖所有质量级别 |
| 填空题智能判题 | 答案归一化函数：去空格/去LaTeX符号/统一大小写/统一括号 |
| 前后端联通 | Vite proxy 配置，开发环境自动转发 /api 请求 |
| CloudStudio 端口暴露 | 使用已有公开端口（35181）运行前端，Vite proxy 转发后端请求 |

### 对话 2：环境运行

**用户**：看的现在弄好的项目怎么运行

**Agent**：
- 后端：`cd backend && pip install -r requirements.txt && python main.py`
- 前端：`cd frontend && npm install && npm run dev`
- 后端自动初始化数据库并填充种子数据

**遇到的问题**：
- `localhost` 在 CloudStudio 云端环境不可直接访问
- 需要通过端口面板暴露端口

---

### 对话 3：端口暴露

**用户**：在端口面板添加前后端并设为公开，你在弄下

**Agent 尝试的方案**：
1. 尝试 CloudStudio API 注册端口 → 不可行
2. 尝试构造预览 URL → 需要手动添加端口
3. 最终方案：将前端迁移到已有的公开端口 35181

**最终架构**：
```
外部访问 → 端口 35181 (Vite) → proxy /api → 后端 8000 (FastAPI)
```

## 核心技术决策记录

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 数据库 | SQLite | 零配置、单文件、适合学习 |
| 后端框架 | FastAPI | 异步支持、自动文档、性能好 |
| 前端框架 | React + Vite | 生态强大、HMR 热更新 |
| UI 库 | Ant Design 6 | 组件丰富、暗色主题支持 |
| 数学公式 | KaTeX | 比 MathJax 更快 |
| 图表库 | ECharts | 中文文档好、效果炫 |
| 间隔算法 | SM-2 | 经典、实现简单、效果明确 |
